Portfolio

ChordShift

A Python-based tool designed to help musicians and composers explore unique harmonic combinations using algorithmic generation. By leveraging pitch-class sets within the 12-tone equal temperament system, ChordShift enables users to discover novel harmonic ideas and expand their musical vocabulary. Compares two search algorithms: A* and Simulated Annealing, to generate intermediate chords that satisfy user-defined harmonic constraints.

Audio Classifier

This project compares the machine learning techniques of Random Forests, Support Vector Machines (SVM), and Recurrent Neural Network with Long Short-Term Memory (RNN-LSTM) in the classification of sounds from the ESC-50 audio dataset.

Arithmetic Practice

Designed to assist a tutor in helping a student build proficiency in fundamental arithmetic operations performed mentally.
